Settings & Instructions

Project Features

  • Home (Robot Face): Displays an animated robot face inspired by WALL‑E.
  • Notes: Create and manage simple notes.
  • Reminders: Add reminders for tasks.
  • Music: Play audio files from a local soundbank.
  • Game: A placeholder for a simple game (more features coming).
  • Arduino Connection: Use the "Connect Arduino" button on the Home page to interact with your Arduino via the Web Serial API (supported on Chromium‑based browsers).

How to Use

Use the navigation bar at the top to switch between features. On the Home page, click "Connect Arduino" to initiate a serial connection with your Arduino. Make sure your Arduino is connected via USB and running a compatible sketch (e.g., one that communicates at 9600 baud).
